## GerenciadorTarefas
Uma aplicação Tkinter que é um Gerenciador de Tarefas

### Permite
- Inserir tarefas, com nome, data, hora e descrição
- Excluir tarefas
- Ordenar tarefas por data e hora
- Salvar agenda e carregar agendas salvas
